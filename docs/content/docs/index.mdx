## Introduction

AgentStack is a comprehensive platform for building, deploying, and managing AI agents. This documentation covers everything you need to get started and make the most of AgentStack.

## Quick Start

Get up and running with AgentStack in minutes:

1. **Create an account** -- Sign up at the AgentStack dashboard
2. **Create your first agent** -- Use the CLI or dashboard to scaffold a new agent
3. **Deploy** -- Push your agent to production with a single command

## Key Concepts

### Agents

Agents are autonomous AI programs that can reason, plan, and take actions to accomplish tasks. AgentStack provides the infrastructure to build and run them reliably.

### Stacks

A stack is a pre-configured combination of tools, models, and frameworks that your agent uses. AgentStack supports multiple stack configurations out of the box.

### Deployments

AgentStack handles the full lifecycle of your agents -- from local development to production deployment with monitoring and observability built in.
